Полагаю что клиент установил удаленный гит репозиторий на своем сервере.

1. Открыть терминал на локальной машине
2. Переместится в рабочую директорию проекта 
    $ cd ~/Projects/test-project
3. Инициализировать пустой репозиторий
    $ git init
3.5. Создать .gitignore (Если нет)
4. Подготовить все файлы файлы к коммиту
    $ git add -A
4. Подтвердить начальный коммит
    $ git commit -m 'Initial commit'
5. Установить данные для подключения к удаленному репозиторию
    $ git remote add origin git@example.com:example/test.git
6. Отправить изменения на удаленный сервер в ветку master. Команда потребует пароля от пользователя git, в дальнейшем, при частом взаимодействии, рекомендую запросить и настроить ssh ключ.
    $ git push -u origin master
